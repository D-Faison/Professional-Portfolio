Overview:

In 2025, I was given the task to create a Movie Finder application that would allow users to select movies based on their IDs, and be able display, add, update and delete movies from the database I created.

Technologies used: MySQL, Java, JSP, HTML, Tomcat and CSS on Apache Netbeans
